

===========
Portability
===========

Chapel is a portable language that is designed to run on a wide variety of hardware and distributed network configurations.

Operating Systems
-----------------

* Linux
* MacOS
* Windows (with WSL)

Compute Architectures
---------------------

* CPU

  * x86

  * ARM

* GPU

  * NVIDIA

  * AMD

Networks
--------

* Slingshot
* Aries
* InfiniBand
* Elastic Fabric Adapter (EFA)
* Ethernet
